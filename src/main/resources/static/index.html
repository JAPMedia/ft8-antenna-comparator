Comparateur d'Antennes FT8 – HB9HIT

Comment utiliser cet outil ?

Cet outil permet aux radioamateurs de comparer les performances de deux ou plusieurs antennes en utilisant les rapports de réception (spots) de PSK Reporter.

  1. Entrez votre Indicatif de réception (ex: HB9HIT).
  2. Connectez votre première antenne, attendez quelques minutes de décodage FT8, puis entrez le nom de l'antenne (ex: "Verticale") et cliquez sur Récupérer les données.
  3. Basculez sur votre deuxième antenne, attendez à nouveau quelques minutes, changez le nom de l'antenne (ex: "Directive") et cliquez sur Récupérer les données.
  4. Analysez les différences de SNR moyen et de nombre de spots dans le tableau récapitulatif.

Note : Les données sont accumulées dans la session. Utilisez "Effacer l'historique" pour recommencer.
